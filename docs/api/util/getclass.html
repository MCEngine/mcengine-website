Java Reflection: Getting Class Object

Code

public static Class getClass(String className) throws ClassNotFoundException {
    // Return the Class object associated with the class name
    return Class.forName(className);
}

Usage

import io.github.mcengine.api.MCEngineApiUtil;

public class Main {
    public static void main(String[] args) {
        try {
            // Example: Get the Class object for "java.util.ArrayList"
            Class<?> clazz = MCEngineApiUtil.getClass("java.util.ArrayList");

            // Print class name to verify
            System.out.println("Class found: " + clazz.getName());
        } catch (ClassNotFoundException e) {
            System.out.println("Class not found: " + e.getMessage());
        }
    }
}

Method Explanation

Method: getClass

This method retrieves the Class object associated with a given class name. It allows dynamic class loading using reflection.

  • Parameters: String className - Fully qualified name of the class.
  • Returns: Class - The Class object representing the specified class.
  • Throws: ClassNotFoundException - If the specified class is not found.
