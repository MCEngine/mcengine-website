Java MySQL getConnection Method

public static Connection getConnection(String host, String database, String username, String password, String port) {
    try {
        return DriverManager.getConnection("jdbc:mysql://" + host + ":" + port + "/" + database, username, password);
    } catch (SQLException e) {
        e.printStackTrace();
    }
    return null;
}
Method: getConnection

This method establishes a connection to a MySQL database.

  • Parameters:
    • String host: The database server's hostname or IP address.
    • String database: The name of the database.
    • String username: The username to connect to the database.
    • String password: The password associated with the username.
    • String port: The port number on which the database server is listening (default for MySQL is 3306).
  • Returns: A Connection object if the connection is successful, otherwise null.
  • Error Handling: Catches SQLException and prints the stack trace if an error occurs during connection.
